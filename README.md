# SCC - Backend 
Contains the NodeJs server code along with MongoDB setup.

### Prerequisites:
* MongoDB
* NodeJs 
* npm
* nodemon

### Setup steps:
* Clone the repository:
`git clone https://TheArhaam@bitbucket.org/jeonjonghyeok5942/scc-backend.git`
* Navigate to folder:
`cd scc-backend/`
* Checkout to develop:
`git checkout develop`
* Install dependencies:
`npm install`
* Start the server:
`nodemon start`
